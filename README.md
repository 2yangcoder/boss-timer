# BOSS计时器 - 多人共享版

## 🎮 功能特色
- ✅ 多人实时数据同步
- ✅ BOSS刷新倒计时
- ✅ 击杀时间记录
- ✅ 数据备份恢复
- ✅ 响应式设计

## 🚀 快速部署

### 方式一：GitHub Pages（推荐）
1. 在GitHub创建新仓库
2. 上传所有文件到仓库
3. 设置 → Pages → 选择main分支
4. 访问生成的URL

### 方式二：Netlify部署
1. 访问 [netlify.com](https://netlify.com)
2. 拖拽整个文件夹到部署区域
3. 获得专属访问链接

### 方式三：直接使用
直接用浏览器打开 `1.html` 文件

## 📱 访问地址
部署完成后，将以下地址分享给团队成员：
- GitHub Pages: `https://你的用户名.github.io/boss-timer`
- Netlify: `https://你的项目名.netlify.app`
